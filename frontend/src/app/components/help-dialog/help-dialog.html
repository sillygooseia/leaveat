Quick Start Guide

people

1. Add Employees

Click the people icon in the toolbar to open the employee panel. Add your team members with their roles, colors, and optional weekly hour limits.

settings

2. Set Coverage Requirements (Optional)

Click the settings icon to configure how many staff members you need each day. Choose from preset templates or customize your own.

add_circle_outline

3. Create Shifts

Click the add_circle_outline button on any day, or double-click anywhere in the day column to quickly add a shift.

content_copy

4. Copy & Duplicate

Use the content_copy menu to copy from last week, copy between days, or clear the entire week.

Key Features

schedule
Automatic Hours Tracking

Weekly hours are calculated automatically. Set max hours per employee to see color-coded warnings.

warning
Conflict Detection

Get warned when scheduling the same person at overlapping times. Shifts with conflicts show a red dashed border.

people
Coverage Requirements

Track whether you have enough staff scheduled for each day. Green = good, amber = approaching limit, red = understaffed.

smart_button
Smart Defaults

When adding shifts, the app remembers each employee's typical hours and suggests them automatically.

check_circle
Create Another

Check "Create another" when adding a shift to keep the dialog open and quickly add multiple shifts.

mouse
Double-Click Shortcut

Double-click any day column to quickly open the add shift dialog.

Shortcuts & Tips

Double-click day Add shift
Click shift Edit or delete shift
Click empty day Add first shift

Pro Tips

  • Use different colors for each employee to make the schedule easier to read at a glance
  • Set weekly hour limits to prevent accidental overwork
  • Copy from last week to save time on recurring schedules
  • Use notes on shifts for special instructions or reminders
  • Coverage requirements help ensure you're never understaffed
  • The "Create another" checkbox is great for bulk shift creation
